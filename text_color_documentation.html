Text Color Documentation

This was by far one of the longest coding assignments I’ve taken on. I originally asked the professor for an example of a code that highlighted text at specific time intervals. It can be found here: Professor’s Original Example.

That example displayed the time signature needed to highlight selected text, which was a great starting point. The problems started when I tried applying this function to multiple lines, the text stayed highlighted instead of reverting, or worse, disappeared altogether due to the “deemphasize” function removing the element entirely.

I got that function from a forum that suggested using `transition` to animate emphasized text, reference here: W3Schools - CSS Transitions.

I browsed Reddit and YouTube for hours trying to figure out how to manage timings as an array. I forget the exact thread, but it helped clarify my approach. Eventually I used `classList.remove` instead of a broken deemphasize function, which solved the problem.

Another issue: my array used IDs like `line1`, `line2`, etc., while the original code used `ele` references, I wasted hours debugging until I caught the mismatch. Lesson: don’t blindly copy code without checking your syntax.

For the video, I downloaded an MP4 version of this incredible edit: Ezio’s Letter (Gaming Speech), which remains one of my favorite narrative moments ever.

I also struggled with centering video until I found this helpful post: How to Center Video in HTML.

Finally, I validated my HTML with this tool during development: HTML Validator.

In the end, there are definitely things I wish I’d known going in, especially around syncing text and clickable jumps in video, but I’ve learned a ton. I even asked the professor about clickable time markers, but after an hour of brain-melting frustration, I decided to move on. Maybe next version!
